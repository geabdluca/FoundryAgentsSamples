# Contoso Product Catalog

## Contoso CloudSuite

Contoso CloudSuite is our flagship enterprise resource planning (ERP) platform, delivered as a fully managed cloud service. It integrates financials, supply chain, human resources, and customer relationship management into a single unified system. CloudSuite is available in three editions: Standard, Professional, and Enterprise.

- **Standard**: Up to 50 users, core financials and HR modules.
- **Professional**: Up to 500 users, adds supply chain, project management, and advanced analytics.
- **Enterprise**: Unlimited users, full module suite, dedicated support SLA, and custom integrations.

Pricing starts at $12 per user per month for Standard. Volume discounts apply for orders of 100+ licenses.

## Contoso Shield (Cybersecurity Platform)

Contoso Shield is a unified cybersecurity platform that provides endpoint detection and response (EDR), cloud security posture management (CSPM), and identity threat protection in a single console. Shield uses AI-driven anomaly detection to surface high-priority threats and reduce alert fatigue.

Key capabilities:
- Real-time endpoint monitoring across Windows, macOS, and Linux
- Automated threat containment and remediation playbooks
- Integration with Microsoft Sentinel, Splunk, and other SIEM platforms
- Compliance reporting for SOC 2, ISO 27001, and NIST CSF

Contoso Shield is licensed per endpoint per month, starting at $8/endpoint.

## Contoso DataLens

DataLens is Contoso's self-service business intelligence and analytics platform. It connects to over 200 data sources, including SQL databases, cloud data warehouses, and SaaS applications, and enables non-technical users to build interactive dashboards without writing code.

Features include drag-and-drop report builder, natural language query (ask a question in plain English and get a chart), scheduled report delivery via email or Teams, and row-level security for data governance.

DataLens is available as a cloud-hosted SaaS or as an on-premises deployment for regulated industries.

## Contoso ConnectEdge (IoT Platform)

ConnectEdge is Contoso's industrial IoT platform designed for manufacturers and logistics providers. It ingests telemetry from sensors, PLCs, and SCADA systems, applies edge ML models for predictive maintenance, and surfaces alerts in a real-time operations dashboard.

ConnectEdge supports MQTT, OPC-UA, and Modbus protocols and can run on any x86 or ARM edge device. Cloud synchronization is available via Contoso CloudSuite or third-party ERP systems.
