Diabetes Dataset — Column Reference
=====================================

Source: Pima Indians Diabetes Dataset (originally from the National Institute of
Diabetes and Digestive and Kidney Diseases).

Population: Female patients of Pima Indian heritage, at least 21 years old.

Target: Binary classification — predicting the onset of diabetes.

Columns
-------

Pregnancies
    Type   : Integer
    Range  : 0–17
    Meaning: Number of times the patient has been pregnant.

Glucose
    Type   : Integer (mg/dL)
    Range  : 0–199
    Meaning: Plasma glucose concentration measured 2 hours into an oral glucose
             tolerance test (OGTT). Higher values indicate poorer glucose
             regulation. Values of 0 represent missing measurements.

BloodPressure
    Type   : Integer (mm Hg)
    Range  : 0–122
    Meaning: Diastolic blood pressure. Values of 0 represent missing measurements.

SkinThickness
    Type   : Integer (mm)
    Range  : 0–99
    Meaning: Triceps skin fold thickness, a proxy for subcutaneous body fat.
             Values of 0 represent missing measurements.

Insulin
    Type   : Integer (mu U/ml)
    Range  : 0–846
    Meaning: 2-hour serum insulin level. Values of 0 represent missing
             measurements and are common in this dataset.

BMI
    Type   : Float (kg/m²)
    Range  : 0.0–67.1
    Meaning: Body Mass Index. Values of 0 represent missing measurements.

DiabetesPedigreeFunction
    Type   : Float
    Range  : 0.078–2.42
    Meaning: A function that scores the likelihood of diabetes based on family
             history. Higher values indicate stronger genetic predisposition.

Age
    Type   : Integer (years)
    Range  : 21–81
    Meaning: Age of the patient in years.

Outcome
    Type   : Integer (binary)
    Values : 0 = No diabetes, 1 = Diabetes
    Meaning: Target label. 1 indicates the patient was diagnosed with diabetes
             within five years of the measurement.

Known Data Quality Issues
--------------------------
Several columns use 0 as a placeholder for missing values where a true zero is
physiologically impossible (Glucose, BloodPressure, SkinThickness, Insulin,
BMI). These should be treated as NaN in any rigorous analysis.

Class Distribution (approximate, full dataset of 768 rows)
------------------------------------------------------------
Non-diabetic (Outcome = 0) : ~65%
Diabetic     (Outcome = 1) : ~35%
